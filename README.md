## Description
This is a To Do App example
## Prerequisites
- Please install tools as bellow: 
    - NodeJS ver 10.x or later
    - Nest CLI: 
    ```bash
    $ npm i -g @nestjs/cli
    ```
    - [MySQL](https://dev.mysql.com/downloads/installer/) Installer 
## How to run this project
- Check out on develop branch for running this project
- Create a new database called "tododb" in the database
## Installation
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
